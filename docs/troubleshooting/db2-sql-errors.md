## Db2 SQL Reference Documentation

The IBM Db2 SQL reference documentation provides comprehensive information about:

- **SQL queries**: SELECT statements, joins, subqueries, and query optimization
- **DDL (Data Definition Language)**: CREATE, ALTER, DROP statements for database objects
- **DML (Data Manipulation Language)**: INSERT, UPDATE, DELETE, MERGE operations
- **Advanced features**: Stored procedures, triggers, user-defined functions, and Db2-specific extensions

Access the official documentation here: [IBM Db2 SQL Documentation](https://www.ibm.com/docs/en/db2/11.5?topic=sql-reference)

## Common SQL Error Scenarios

When working with SQL in the Db2 Developer Extension, you might encounter errors related to:

- **Syntax errors**: Incorrect SQL statement structure or missing keywords
- **Object not found**: References to tables, views, or columns that don't exist
- **Permission errors**: Insufficient privileges to execute certain operations
- **Data type mismatches**: Incompatible data types in comparisons or assignments
- **Constraint violations**: Primary key, foreign key, or check constraint violations

The IBM Db2 SQL reference documentation provides detailed syntax diagrams, examples, and explanations to help you resolve these issues.
