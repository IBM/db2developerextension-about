# {{ page.title }}

 Your SQL queries can be saved to a file. 
 
 
 To save the queries to a file:


 1. Write your queries in the Editor and click **Run**. 
 2. Go to the **Result** pane and click the **Save** icon.
 3. Enter the location and name of the file when prompted.
 4. Press **Enter** to save the queries to the specified location.
