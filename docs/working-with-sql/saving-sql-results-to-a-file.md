# {{ page.title }}

 Your SQL statements can be saved in .sql or .db2.sql format. 
 
 
 To save the statements to a file:


 1. Write your statements in the editor and click **Run**. 
 2. Go to the **Result** pane and click **Save**.
 3. Enter the location and name of the file when prompted.
 4. Press **Enter** to save the statements to the specified location.
